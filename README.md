# Python 練習專案

這個repository包含了各種Python練習題目的解答，涵蓋了基礎程式設計概念和演算法。

## 檔案說明

- `1.py` - 基礎練習
- `Email判斷.py` - Email格式驗證
- `反轉字串0.py` - 字串反轉練習
- `愛找麻煩的老師0.py` - 邏輯練習題
- `找中間數0.py` - 尋找中位數演算法
- `找零機器0.py` - 找零錢演算法
- `樂透號碼判斷0.py` - 樂透號碼驗證
- `複利0.py` - 複利計算
- `豹子0.py` - 數字模式判斷
- `質數.py` - 質數判斷演算法
- `迴文判定.py` - 迴文字串判斷
- `金彩5390.py` - 特定數字遊戲

## 使用方法

每個檔案都可以獨立執行：

```bash
python 檔案名稱.py
```

## 學習目標

- 條件判斷和迴圈
- 函數設計
